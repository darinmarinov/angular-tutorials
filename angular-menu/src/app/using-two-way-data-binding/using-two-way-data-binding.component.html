Използванe на Two-Way Data Binding

Two-way data binding комбинира input и output свързването в една единствена нотация, използвайки ngModel директива 

<input [(ngModel)]="name" >

Друг по очевиден пример затова

<input [(ngModel)]="username">
<p>Hello {{username}}!</p>

В този пример каквото напишем в input тага ще се отрази в username в параграфа.

Как изглежда всъщност two-way data binding, затази цел трябва да погледнем по подробно input тага и да направим някой промени 

<input [value]="username" (input)="username = $event.target.value">

<p>Hello {{username}}!</p>

Нека погледнем какво се случва по-отблизо всъщност

  • [value]=”username” свързва израза username със стойноста на input свойството.
  • (input)=”expression”
  • username = $event.target.value този израз започва да се изпълнява веднага след като започне да се пише в полето.
  • $event

За да създадем наш компонент които да поддържа two-way data binding, трябва да дефинирам @Output и @Input свойства.
